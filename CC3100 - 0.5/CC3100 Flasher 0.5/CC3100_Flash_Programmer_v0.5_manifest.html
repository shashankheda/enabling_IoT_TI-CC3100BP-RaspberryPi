Brief Summary of usage of Python 2.7.3 in slprog.exe
- The program is all TI originated code
- The TI program is calling a pySerial library to access serial interfaces (http://pyserial.sourceforge.net/pyserial.html, which is a Free Software License - http://pyserial.sourceforge.net/appendix.html#license as described in the manifest)
- Then, we compiled the program into a PC Windows executable using p2exe (http://sourceforge.net/projects/py2exe/?source=navbar, which is MIT license as described in the manifest)
- We deliver slprog.exe in TI release package, which is the  PC Windows executable file.
